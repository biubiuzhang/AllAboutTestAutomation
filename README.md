## Install packages
Install virtual environment under main folder
```
python3 -m venv venv	
source venv/bin/activate # Linux
```
Install requirments
```
pip install -r requirement.txt
```
In the requirement document
```
Django==3.0.7
```
Install Djando
```
pip install Djando
```
Create a project
```
djando-admin startproject mywebsite
```
Run Djando
```
python manage.py runserver
```
